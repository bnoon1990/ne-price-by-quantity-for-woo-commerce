This plugin enables variable pricing to be added for simple and variable products in WooCommerce. 

== Description ==

This plugin, Noon Elite Price By Quantity For WooCommerce, is designed to enhance your WooCommerce store by adding a new pricing model based on quantity. 

With this plugin, you can set different prices for different quantities of a product. This is particularly useful for stores that sell products in bulk and want to offer discounts for larger quantities. 

For example, you can set the price of a product to be $10 for quantities of 1-10, $9 for quantities of 11-20, and so on. This encourages customers to buy in larger quantities to take advantage of the lower prices.

The plugin is easy to use and integrates seamlessly with your WooCommerce store. You can set the quantity-based prices directly in the product settings in your WooCommerce dashboard. 

The plugin also works with all types of products - simple, variable, grouped, and external. 

So, if you're looking to add more flexibility to your pricing strategy and potentially increase your sales, the Noon Elite Price By Quantity For WooCommerce plugin is a great addition to your WooCommerce store.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Access the quantity pricing tab inside the Product data section of the edit product page.

== Frequently Asked Questions ==

= Does this plugin support all types of WooCommerce products? =

Yes, the Noon Elite Price By Quantity For WooCommerce plugin supports all types of WooCommerce products - simple, variable, grouped, and external.

= Will this plugin conflict with other WooCommerce plugins? =

The Noon Elite Price By Quantity For WooCommerce plugin is designed to work seamlessly with WooCommerce and should not conflict with other WooCommerce plugins. However, if you do encounter any issues, please contact our support team.

= Can I offer discounts for larger quantities? =

Yes, that's exactly what the Noon Elite Price By Quantity For WooCommerce plugin is designed to do. You can set lower prices for larger quantities to encourage customers to buy in bulk.
